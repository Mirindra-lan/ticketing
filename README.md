# Ticketing

## Description:
Ticketing is an automated log management and ticket generation system designed to streamline error tracking and IT service management. The project leverages Docker, AI, and GLPI integration to simplify the workflow from log detection to ticket creation.

## Key functionnalities:
**1. Read logs from Docker container**
**2. Transform logs into tickets using an LLM API**
**3. Create tickets in GLPI via GLPI API**

## Technologies used:
-Node.js for scripting and API integration
-Docker for containerized applications
-Large Language Model(LLM) API for natural language processing
-GLPI API for ticket management
